# Jeseem's Cinematic Portfolio 🚀

Welcome to the open-source repository for my personal developer portfolio. This project was engineered from the ground up to push the boundaries of modern web application interactions using a purely React/Next.js stack, effectively completely erasing the need for heavyweight WebGL libraries by utilizing raw physics-based UI components.

## 🌟 Live Features

1. **Intelligent Dynamic Layout Navigation**  
   - An advanced `IntersectionObserver` continually sweeps the user's viewport, synchronizing the active application state perfectly across hidden command palettes and lower docks.
   - 60+ FPS high-end physics-spring layout-id transitions for Project Modals.
   
2. **Glassmorphic UI Systems**
   - 10+ layers of deeply optimized CSS backdrops and gradient blurring.
   - Tailored `TiltCard.tsx` components for intense interactive depth.

3. **Context-Aware Interactions**
   - Implemented a custom canvas/cursor engine that binds dynamically to the physical objects scaling across the viewport.

## 🛠️ Stack Architecture

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router & Server Actions)
- **Styling Matrix:** [Tailwind CSS](https://tailwindcss.com/)
- **Animation Engine:** [Framer Motion](https://www.framer.com/motion/) natively deployed for physics layouts
- **Asset/Iconography:** Lucide Icons

## 🚀 Quick Start

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Jeseem24/portfolio.git
   cd portfolio
   ```

2. **Install modules:**
   ```bash
   npm install
   ```

3. **Start exactly where I left off:**
   ```bash
   npm run dev
   ```

Navigate your browser to `http://localhost:3000` to interact with the raw UI!
